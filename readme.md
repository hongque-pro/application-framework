业务开发基础结构，以统一的技术栈快速开始一个应用。

### 和 infra 系列 jar 包区别：

- infra 偏向底层业务无关的架构实现类库
- application framework 偏向于应用层开发的通用结构，少量的业务直接重用
- application framework 依赖于 infra, 在更接近业务的层次来简化代码
